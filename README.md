# Dew Framework

A modern PHP framework built from scratch.

## Requirements
- PHP 8.0 or higher
- Composer

## Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/my-framework.git

# Install dependencies
cd my-framework
composer install

# Start development server
php -S localhost:8000 -t public
```
